Onebot Command Routing

Using the knowledge of software engineering, data structures and, compiler front end. I code a high-performance command router by preparing the parameter type before starting to receive messages and manage all commands into a tiered tree.

This project is an important part of VolcEnging ImageX recommended on VolcEngins's website.

Miscellaneous

Onebot Concept Collaborator

Onebot Concept Collaborator github/howmanybots/onebot

An adapter concept about chatting robots. Providing a similar API between various kinds of chatting robot platforms for easy migration from this one to another one.

The main idea of this protocol is building a set of adapters to make many kinds of IM chatting robot systems expose a certain commonly used API.

This project is in the list of Summer 2021 of Open Source Promotion Plan, which is held by the Institute of Software Chinese Academy of Sciences
